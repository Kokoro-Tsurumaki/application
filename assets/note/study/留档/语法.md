送假名是​**​标注在汉字后面的假名​**​，用于指示动词/形容词的活用形或读音
**核心作用​**​：

- ​**​明确汉字发音​**​（尤其训读词）。
- ​**​标示语法功能​**​（如动词时态、否定形等）。

1. いくつも（幾つも）
    
    - 疑问词"いくつ"（几个）＋强调助词"も"
    - 表示数量多，"许多个"的意思
2. ゆめ（夢）
    
    - 名词，"梦想"的意思
3. を（格助词）
    
    - 表示动作的对象
    - 这里指"数える"的对象是"夢"
4. かぞえても（数えても）
    
    - 动词"数える"（数数）的て形＋も
    - "ても"表示让步关系，"即使...也..."

|        |     |        |
| ------ | --- | ------ |
| は（作助词） | wa  | 语法功能标记 |
| は（单词内） | ha  | 音节组成部分 |
