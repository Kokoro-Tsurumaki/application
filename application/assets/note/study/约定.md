周计划

- ​**基础谷物：​**​
    - ​**大米：​**​ 约50-80克 (约半杯)。提供汤的稠滑基底。
    - ​**绿豆：​**​ 20-30克。性凉，清热利湿，不宜过多以免汤性过寒。
    - ​**薏仁米：​**​ 20-30克。利水渗湿，健脾。
- ​**坚果与果实：​**​
    - ​**红枣：​**​ 5-8颗，洗净后可去核（不去核也可，但喝时注意）。增加天然甜味，补中益气。
    - ​**枸杞：​**​ 10-15克。​**切记不可久煮**，最后放。
    - ​**芝麻：​**​ 10-15克（黑白皆可）。​**强烈建议使用熟芝麻**，并在最后撒入，香气更浓。
- ​**蔬菜：​**​
    - ​**萝卜：​**​ 白萝卜或胡萝卜100-150克（约半根），去皮切块或滚刀块。
