
のびた人陰を舗道にならべ
拉长的身影 并列在柏油路

- 1、 のびた（伸びた）​​

| 属性       | 说明                    |
| -------- | --------------------- |
| ​**原形**​ | 伸びる（のびる）              |
| ​**词性**​ | 自动词 → ​**た形作定语修饰名词**​ |
| ​**意思**​ | "被拉长的"（修饰「人陰」）        |
| ​**变化**​ | 伸びる → 伸びた（过去分词作形容词）   |


- 2、  人陰（じんかげ）​​

| 属性       | 说明                      |
| -------- | ----------------------- |
| ​**构成**​ | 人（じん）＋陰（かげ）             |
| ​**意思**​ | "人影"（汉字直译）              |
| ​**注意**​ | 日常多用「人影（ひとかげ）」，此处为诗歌化表达 |

- 3、を​

| 属性       | 说明                      |
| -------- | ----------------------- |
| ​**助词**​ | 表示**动作对象**​（英语"object"） |
| ​**功能**​ | 标记宾语（此处宾语是「人陰」）         |
| ​**发音**​ | 永远读 "o"（类似中文"哦"）      



#### **① ならべ（並べ）​**​

| 属性       | 说明                  |
| -------- | ------------------- |
| ​**原形**​ | 並べる（ならべる）           |
| ​**词性**​ | 他动词（需带宾语）           |
| ​**意思**​ | "排列、摆放"             |
| ​**变化**​ | 命令形或省略形（日常口语常省略「る」） |
  |



#### **⑤ 舗道に（ほどうに）​**​

|属性|说明|
|---|---|
|​**词汇**​|舗道（ほどう）="铺装的道路"（人行道）|
|​**助词**​|に（表示**动作地点**）|
|​**意思**​|"在道路上"|
