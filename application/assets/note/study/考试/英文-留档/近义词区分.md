### will、should、shall
|            | ​**will**​         | ​**should**​       | shall            |
| ---------- | ------------------ | ------------------ | ---------------- |
| ​**核心含义**​ | ​**未来必然性**​（客观/主观） | ​**义务/建议**​（道德/逻辑） | **必然性/正式性**（强制性强 |
| ​**语气**​   | 中性或主动承诺            | 劝告、推测或遗憾           | 正式规则或法律文件        |
| ​**强制程度**​ | 无强制（单纯描述未来）        | 弱强制（"最好这样做"）       | 强制               |

| ​​         | **each**​​        | **every**         |
| ---------- | ----------------- | ----------------- |
| ​**侧重点**​  | ​**个体性**​（强调单个成员） | ​**整体性**​（强调全部成员） |
| ​**数量范围**​ | 通常用于 ​**2个或以上**​  | 通常用于 ​**3个或以上**​  |
| ​**语法搭配**​ | 可作代词、形容词、副词       | 仅作形容词             |

| ​**单词**​       | ​**本质含义**​   | ​**隐含情感**​       |
| -------------- | ------------ | ---------------- |
| ​**normal**​   | 符合标准、常规或自然状态 | 中性或积极（强调“应然状态”）  |
| ​**ordinary**​ | 普通、平凡，无特殊之处  | 中性或轻微消极（强调“无特色”） |

|特性|Therefore|Thus|
|---|---|---|
|​**核心含义**​|​**因此，所以**。侧重于**逻辑推理和结论**，是“因为A，所以B”中的“所以”。|​**以此方式，从而**。侧重于**方式、方法或结果**，是“通过A，从而得到B”中的“从而”。|
|​**语气**​|非常正式，常用于学术论文、数学证明、法律文书和严谨的论证中。|正式，但比 `therefore` 稍弱，更偏重书面语。|
|​**侧重点**​|​**因果关系**​ (Causality)|​**方式或结果**​ (Manner or Result)|
|​**常见位置**​|常用于引出一个结论，位置灵活（句首、句中、与 `and` 连用）。|常用来描述一个过程如何导致一个结果，多用于句中。|
