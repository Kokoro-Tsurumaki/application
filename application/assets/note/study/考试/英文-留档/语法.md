## 五大类谓语动词 五大句型

1. 无需承受者(宾语)的动词
	The rabbit slept.


### **一、形容词 → 名词**​

#### 1. ​**规则后缀**​

| 后缀           | 规则                         | 示例                     |
| ------------ | -------------------------- | ---------------------- |
| ​**​-ness**​ | 直接添加（去 `-y` 变 `-i`）        | happy → happiness      |
|              |                            | dark → darkness        |
| ​**​-ity**​  | 去 `-able/-ible` 或 `-al` 后加 | possible → possibility |
|              |                            | national → nationality |
| ​**​-ty**​   | 部分形容词固定使用                  | safe → safety          |
|              |                            | cruel → cruelty        |
| ​**​-th**​   | 词干变化 + `-th`               | long → length          |
|              |                            | strong → strength      |
| ​**​-ce**​   | 去 `-t` 加 `-ce`             | patient → patience     |
|              |                            | innocent → innocence   |

#### 2. ​**不规则变化**​

|形容词|名词（无固定规则）|
|---|---|
|good|goodness|
|high|height（非 *highness）|
|hot|heat|
|broad|breadth|

---

### ​**二、动词 → 名词**​

#### 1. ​**规则后缀**​

|后缀|规则|示例|
|---|---|---|
|​**​-tion**​|动词以 `-te` 结尾|create → creation|
|​**​-sion**​|动词以 `-de`/`-se` 结尾|decide → decision|
|||confuse → confusion|
|​**​-ment**​|直接添加|develop → development|
|||govern → government|
|​**​-ance**​|动词以 `-y` 结尾|rely → reliance|
|​**​-ence**​|动词以 `-e` 结尾|exist → existence|
|​**​-al**​|直接添加|arrive → arrival|
|​**​-ing**​|动名词形式|swim → swimming|

#### 2. ​**不规则变化**​

|动词|名词（无固定规则）|
|---|---|
|speak|speech|
|choose|choice|
|break|breach/breakage|

---

### ​**三、名词 → 形容词**​

#### 1. ​**规则后缀**​

|后缀|规则|示例|
|---|---|---|
|​**​-ful**​|直接添加|beauty → beautiful|
|​**​-less**​|表示“缺乏”|hope → hopeless|
|​**​-ous**​|名词以 `-y` 结尾变 `-i`|danger → dangerous|
|||glory → glorious|
|​**​-al**​|直接添加|nature → natural|
|​**​-ic**​|名词以 `-y` 结尾变 `-etic`|history → historic|
|||economy → economic|
|​**​-able**​|表“可...的”|fashion → fashionable|

#### 2. ​**不规则变化**​

|名词|形容词（无固定规则）|
|---|---|
|child|childish（非 *childy）|
|friend|friendly|

---

### ​**四、动词 → 形容词**​

|后缀|规则|示例|
|---|---|---|
|​**​-able**​|直接添加|wash → washable|
|​**​-ible**​|固定搭配（拉丁词源）|access → accessible|
|​**​-ive**​|动词以 `-de`/`-se` 结尾|decide → decisive|
|||impress → impressive|
|​**​-ent**​|动词以 `-er` 结尾|differ → different|
|​**​-ant**​|动词以 `-y` 结尾|occupy → occupant|

---

### ​**五、名词 → 动词**​

|后缀|规则|示例|
|---|---|---|
|​**​-ize**​|美式英语常用|modern → modernize|
|​**​-ise**​|英式英语常用|advert → advertise|
|​**​-ify**​|名词以辅音结尾|beauty → beautify|
|​**​-en**​|名词为具体物体|strength → strengthen|

---

### ​**六、形容词 → 副词**​

|后缀|规则|示例|
|---|---|---|
|​**​-ly**​|直接添加|quick → quickly|
||以 `-y` 结尾变 `-i`|happy → happily|
||以 `-le` 结尾去 `e` 加 `-y`|gentle → gently|
|​**​-wise**​|表“方式”|clock → clockwise|

---

### ​**七、特殊发音变化**​

#### 1. ​**重音移位**​

|词性变化|示例|发音变化|
|---|---|---|
|动词 → 名词（-tion）|produce → production|/prəˈdjuːs/ → /prəˈdʌkʃən/|
|名词 → 形容词（-ic）|democrat → democratic|/ˈdeməkræt/ → /ˌdeməˈkrætɪk/|

#### 2. ​**辅音变化**​

|变化类型|示例|
|---|---|
|/t/ → /ʃ/|act → action|
|/d/ → /ʒ/|decide → decision|
|/s/ → /ʃ/|press → pressure|

---

### ​**八、完全无规则变化**​

|原词（动词）|名词|形容词|
|---|---|---|
|see|sight|visible|
|die|death|dead|
|go|journey|gone|

---

### ​**总结规律**​

1. ​**规则变化**​：
    - `-ness`（形→名）、`-ly`（形→副）、`-ize`（名→动）等后缀高度可预测。
2. ​**半规则变化**​：
    - `-tion/-sion`（动→名）依赖动词词尾音素。
3. ​**不规则变化**​：
    - 高频词常不规则（如 `long → length`），需单独记忆。
优先掌握 `-ness, -tion, -ful, -ly` 四大后缀，覆盖80%日常词汇。
