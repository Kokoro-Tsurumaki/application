|                  |               |                            |
| ---------------- | ------------- | -------------------------- |
| `relative`       | 建立新的定位上下文     | `position: relative`       |
| `absolute`       | 使元素脱离文档流      | `position: absolute`       |
| `inset-0`        | 四边距离设为0，撑满父容器 | `top/right/bottom/left: 0` |
| `flex`           | 启用弹性布局        | `display: flex`            |
| `justify-center` | 水平居中          | `justify-content: center`  |
| `items-center`   | 垂直居中          | `align-items: center`      |
flex 会分配最大空间给子项
