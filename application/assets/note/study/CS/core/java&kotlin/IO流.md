## Buffer流 vs 普通流性能对比

​**是的，Buffer流通常比普通流更快**，但具体性能提升取决于使用场景：

|特性|普通流|Buffer流|性能影响|
|---|---|---|---|
|系统调用次数|每次读写都调用|批量读写减少调用|⭐⭐⭐⭐⭐|
|磁盘寻道时间|频繁寻道|顺序读写减少寻道|⭐⭐⭐⭐|
|CPU使用率|高（频繁切换）|低（批量处理）|⭐⭐⭐|
|内存使用|低|较高（需要缓冲区）|⭐|
|小文件处理|无优势|无优势|-|
|大文件处理|慢|快（2-10倍）|⭐⭐⭐⭐|

|​**分类**​|字节流|字符流|
|---|---|---|
|​**输入**​|InputStream|Reader|
|​**输出**​|OutputStream|Writer|
|​**缓冲**​|BufferedInputStream|BufferedReader|
|​**转换**​|-|InputStreamReader|
