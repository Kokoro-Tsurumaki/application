## 图像

### 图像色彩模式
1. 位图模式：
	图像只有黑与白，占用控件最小，相当于只有0或1，一幅彩色图如果要转换成位图模式一般不能直接转换，首先需要将图像转换灰度模式
2. 灰度模式:
	即使用单一色调表示图像，与位图不同，不只有0和1，使用256级的灰度来表示图像，一个像素相当于占用8为一个字节，每个像素值使用0到255的亮度值代表，其中0为黑色，255为白色，相当于黑->灰->白的过度。相对于位图模式，能表现出一定细节，占用空间也比位图较大
3. RGB模式:
	为真色彩，也是最常见的，有三个颜色通道，分为红、绿和蓝，每个都占用8位一个字节来表示颜色信息，RGB模式表现出的细节更加明显
4. HSV模式:
	根据日常生活中视觉对色彩的观察得到的色彩模式，所有属性都是由色彩三属性来表述
	- H(色相)：指从物体反射或透过物体传播的颜色
	- S(饱和度)：指颜色的强度或纯度，表示色相中灰色成分所占比例
	- V(亮度)：指颜色对相对明暗的程度，通常100%定义为白色；0%为黑色
此外，还有CMYK模式和YUV模式等。

### Imgproc.cvtColor
COLOR_BGR2RGB 颜色空间转换
COLOR_BGR2GRAY BGR转换为灰度
COLOR_GRAY2RGB 灰度转为RGB
COLOR_RGB2HSV RGB转为HSV
COLOR_RGB2RGBA 添加alpha通道

### 1. `Utils.matToBitmap()` 的格式要求

- ​**支持的格式**​：
    
    - 单通道：`CV_8UC1` (灰度图)
    - 三通道：`CV_8UC3` (BGR 顺序)
    - 四通道：`CV_8UC4` (BGRA 顺序)
- ​**不支持的格式**​：
    
    - RGB 顺序的三通道 Mat
    - HSV 格式的任何 Mat
    - 其他非标准格式
