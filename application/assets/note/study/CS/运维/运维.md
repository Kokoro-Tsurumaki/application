### nohup
```
# 使用 nohup 在后台运行，即使退出 SSH 也保持运行
nohup java -jar your-app.jar > app.log 2>&1 &

# 查看进程
ps aux | grep java

# 查看日志
tail -f app.log

# 停止应用
kill <PID>
```
## 生产环境部署（推荐）
#### 步骤 1：创建专用用户

```

# 创建专门运行应用的用户
sudo useradd -r -s /bin/false ktor-app

# 创建应用目录
sudo mkdir -p /opt/ktor-app
sudo chown ktor-app:ktor-app /opt/ktor-app
```
#### 步骤 2：部署应用文件


```
# 上传 jar 文件到应用目录
sudo cp your-app.jar /opt/ktor-app/

# 设置权限
sudo chown ktor-app:ktor-app /opt/ktor-app/your-app.jar
sudo chmod 500 /opt/ktor-app/your-app.jar

# 创建日志目录
sudo mkdir /var/log/ktor-app
sudo chown ktor-app:ktor-app /var/log/ktor-app
```

#### 步骤 3：创建 Systemd 服务文件


```
sudo nano /etc/systemd/system/ktor-app.service
```
